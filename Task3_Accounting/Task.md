📊 Практична задача: Бухгалтерія етно-ярмарку
📌 Огляд
Після успішного ярмарку хендмейд-виробів, майстерні потрібно підбити підсумки продажів. Твоє завдання — розробити програму для аналізу виручки, використовуючи масиви та цикли.

🛠 Технічні вимоги
1. Створення масиву (Arrays)
Створи масив цілих чисел int[] sales розміром на 5 елементів.

2. Введення даних (for loop)
Використовуючи цикл for, реалізуй послідовне запитування цін у користувача.

Кожна введена ціна має бути збережена у відповідний індекс масиву.

Порада: Використовуй int.TryParse, щоб програма не зламалася, якщо замість ціни введуть текст.

3. Обчислення результатів (foreach або for)
Проаналізуй масив, щоб знайти два ключові показники:

Загальна сума: Додай усі значення масиву в одну змінну totalSum.

Найдорожчий товар: Знайди найбільше число в масиві та збережи його в maxPrice.

4. Фінальний звіт
Виведи результати у консоль у зручному вигляді:

Plaintext
=== ЗВІТ ЗА ДЕНЬ ===
Усі продажі: [ціна1, ціна2, ціна3, ціна4, ціна5]
Загальна виручка: XXX грн.
Найдорожчий виріб: XXX грн.
====================
💡 Поради від Ментора
[!IMPORTANT] Пам'ятай, що масив має фіксовану довжину. Якщо ти створила масив на 5 елементів, останній доступний індекс буде 4, оскільки відлік починається з 0.

Шпаргалка з алгоритму пошуку максимуму:
C#
int max = sales[0]; // Припускаємо, що перший елемент - найбільший
foreach (int price in sales)
{
    if (price > max) 
    {
        max = price; // Якщо знайшли більше число, оновлюємо наш максимум
    }
}
✅ Критерії виконання
[ ] Масив ініціалізовано з фіксованим розміром.

[ ] Цикл for використовується для заповнення масиву.

[ ] Значення суми та максимуму обчислюються коректно.

[ ] Код не "падає" при введенні некоректних символів (використано TryParse).
