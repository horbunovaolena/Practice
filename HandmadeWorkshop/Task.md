# 📝 Практична задача: Аналізатор товарних бирок

## 📌 Огляд
Ти розробляєш модуль для системи обліку етно-майстерні. Твоє завдання — реалізувати перевірку артикулів (кодів товарів) та автоматичну обробку описів виробів, використовуючи сучасні методи роботи з рядками в C#.

---

## 🛠 Технічні вимоги

### 1. Валідація артикула (`StartsWith`, `Length`)
Артикул має суворий формат: `ETHNO-XX-YYYY` (наприклад, `ETHNO-OR-2026`).
* **Дія:** Запитай у користувача артикул.
* **Перевірка 1:** Перевір, чи починається рядок з префікса `ETHNO-`.
* **Перевірка 2:** Перевір, чи загальна довжина рядка становить рівно **13 символів**.
* **Результат:** Якщо хоча б одна умова не виконана, виведи: *"Помилка: Артикул не відповідає стандарту!"*

### 2. Розшифровка коду (`Split`)
Якщо артикул правильний, розбий його на три логічні частини.
* **Механіка:** Використай дефіс `-` як роздільник.
* **Виведи результат:**
    - Префікс: [частина 1]
    - Код категорії: [частина 2]
    - Рік виготовлення: [частина 3]

### 3. Обробка опису товару (`Trim`, `Replace`)
Майстри часто поспішають, тому описи потребують автоматичного виправлення.
* **Дія:** Запитай у користувача повний текст опису.
* **Очищення:** Видали зайві пробіли на початку та в кінці (`Trim`).
* **Заміна:** Знайди всі згадки слова `"пластик"` (у будь-якому регістрі) та заміни їх на `"еко-матеріал"`.

### 4. Контроль безпеки (`Contains`)
* **Логіка:** Перевір, чи містить опис слова `"золото"` або `"срібло"`.
* **Результат:** Якщо містить, виведи повідомлення: *"🔔 Увага! Товар містить дорогоцінні метали. Потрібне додаткове страхування."*

## 💡 Поради від Ментора

> [!IMPORTANT]
> Метод `Split` повертає масив рядків (`string[]`). Щоб отримати елементи, звертайся до них за індексами: `parts[0]`, `parts[1]`, `parts[2]`.

### Приклад обробки рядка:
```csharp
string rawDescription = "  Цей виріб містить пластик.  ";
string cleanDescription = rawDescription.Trim().Replace("пластик", "еко-матеріал");
